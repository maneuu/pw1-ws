# 📚 PW1 - Programação Web 1

Repositório destinado às atividades da disciplina de Programação Web 1.

## 🧠 Tecnologias

- TypeScript
- Node.js

---

## 📁 Estrutura do Projeto

Cada atividade está organizada em sua própria pasta:

```
/atividade-01
/atividade-02
/atividade-03
...
```

Cada pasta pode possuir arquivos e implementações diferentes.

---

## 🚀 Como executar uma atividade

### 1. Acesse a pasta da atividade desejada

```bash
cd 01-.....
```

---

### 2. Instale as dependências (se necessário)

```bash
npm install
```

---

### 3. Compilar o TypeScript

```bash
npx tsc
```

---

### 4. Executar o arquivo principal

O nome do arquivo pode variar de acordo com a atividade, por exemplo:

```bash
node index.js
```

ou

```bash
node main.js
```

ou outro arquivo `.js` presente na pasta.

---

## ⚡ Executar sem compilar (opcional)

```bash
npx ts-node arquivo.ts
```
